# Car Rental Management System

## 1. Обзор проекта

Car Rental Management System — это приложение для управления арендой автомобилей, которое позволяет клиентам просматривать доступные автомобили, оформлять аренду и следить за историей обслуживания. Система поддерживает управление данными о клиентах, автомобилях, классах автомобилей, арендных соглашениях и обслуживаниях.

## 2. Ключевые функции

- **Управление автомобилями**: Добавление, обновление и удаление информации об автомобилях.
- **Управление клиентами**: Регистрация клиентов и управление их данными.
- **Арендные соглашения**: Оформление арендных соглашений и отслеживание их состояния.
- **Обслуживание автомобилей**: Учет записей о техническом обслуживании.
- **Отчеты**: Генерация отчетов по клиентам, автомобилям и арендным соглашениям.

## 3. Системные требования

- **.NET SDK** (версии 8 или выше)
- **SQL Server** (LocalDB или другая версия)
- **Visual Studio 2022** с установленным рабочим набором для разработки на C#.
- **Windows, Linux или macOS**
